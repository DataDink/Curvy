Html Service

The html service contains a small number of common functionalities used frequently by Curvy. 

   Application.extend(['html', function(html) {
      console.log(html.encode('<div>HTML Content</div>'));
   }])

encode(html) Escapes a string so that it can be safely displayed inside markup.

parse(html) Parses a string into an array of document nodes.

getInputValue(element) Tries to find and return a relevant value based on the element type.

setInputValue(element, value) Tries to set the relevant value based on the element type.

setStyle(selector, css) Adds a css rule to the document.

removeClass(element, class) Removes the class from the element and leaves other classes behind

addClass(element, class) Adds the class to the element replacing any exiting duplicates.

hasClass(element, class) Returns true if the element has the specified class

toggleClass(element, class) Adds or removes the class from the element.

formatParameters(object) Converts an object into a uri parameter string.

formatUri(uri, parameters) Adds or combines parameters to an existing uri.

parseQuery(parameterString) Parses the parameter portion of a uri into an object.
