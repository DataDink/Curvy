data-template

Similar to Angular's ng-repeat this converts an element into a template and binds it to a value or collection of values on the view-model. 

	<div data-template="searchResults">
		<span data-format="{{name}}"></span>
	</div>

	app.viewmodel('my-vm', function() {
		this.searchResults = [];
			
		for (var i = 0; i < 10; i++) {
			this.searchResults.push({name: 'item #' + i});
		}
	});
