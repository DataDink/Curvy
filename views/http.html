Http Service

The http service is a small wrapper around XHR (ajax) that provides some common short-cuts for configuring restful calls to an API.

http.setGlobalHeader(headerName, headerValue) can be used to set a header on all requests for the life of the application.

http.setGlobalConfig(configName, configValue) can be used to set a configuration option on all requests for the life of the application.

http.setGlobalParam(name, value) can be used to persist a value in all requests for the life of the application.

http.setGlobalQuery(name, value) can be used to persist a query-only value in all requests for the life of the application.

http.setHeader(headerName, headerValue) can be used to set a header on all requests for the life of this instance of the HTTP service.

http.setConfig(configName, configValue) can be used to set a configuration option on all requests for the life of this instance of the HTTP service.

http.setParam(name, value) can be used to persist a value in all requests for the life of this instance of the HTTP service.

http.setQuery(name, value) can be used to persist a query-only value in all requests for the life of this instance of the HTTP service.

http.get(uri, params, onSuccess, onError, headers, config) sends a get request to the uri.
uri: the uri to send the get request to.
params: an object containing the request values to send.
onSuccess: a callback that will fire when the request succeeds.
onError: a callback that will fire when the request fails.
headers: an object with the header to set for this request.
config: an object with the configuration options for this request.

http.delete(uri, params, onSuccess, onError, headers, config) sends a delete request to the uri.
uri: the uri to send the get request to.
params: an object containing the request values to send.
onSuccess: a callback that will fire when the request succeeds.
onError: a callback that will fire when the request fails.
headers: an object with the header to set for this request.
config: an object with the configuration options for this request.

http.post(uri, body, onSuccess, onError, headers, config) sends a post request to the uri.
uri: the uri to send the get request to.
body: an object containing the request values to send.
onSuccess: a callback that will fire when the request succeeds.
onError: a callback that will fire when the request fails.
headers: an object with the header to set for this request.
config: an object with the configuration options for this request.

http.patch(uri, body, onSuccess, onError, headers, config) sends a get request to the uri.
uri: the uri to send the get request to.
body: an object containing the request values to send.
onSuccess: a callback that will fire when the request succeeds.
onError: a callback that will fire when the request fails.
headers: an object with the header to set for this request.
config: an object with the configuration options for this request.

http.put(uri, body, onSuccess, onError, headers, config) sends a get request to the uri.
uri: the uri to send the get request to.
body: an object containing the request values to send.
onSuccess: a callback that will fire when the request succeeds.
onError: a callback that will fire when the request fails.
headers: an object with the header to set for this request.
config: an object with the configuration options for this request.

configuration: The following configuration options are available:
async: Set to false if you want the request to not be async
user: Sets a username on the request
pass: Sets a password for the username on the request
mimeType: Overrides the mimeType for the request
withCredentials: Set to true to force credentials with the request
timeout: Sets a timeout for the request

this: On a success or error callback the "this" keyword will be set to an object with request/response info.

Example Usages: 

	// A huge configuration...
	app.viewmodel('my-viewmodel', ['http', function(http) {
		http.get(
			'http://my-api/something',
			{ somethingId: 1, anotherId: 2 },
			function(data) {
				console.log('success');
				console.log(this);
				console.log(data);
			},
			function(data) {
				console.log('error');
				console.log(this);
				console.log(data);
			},
			{ 'SomeHeader': 'SomeHeaderValue' },
			{ withCredentials: true }
		);
	}]);
	
	// A typical configuration...
	app.viewmodel('my-viewmodel', ['http', function(http) {
		http.get(
			'http://my-api/something', 
			{ id: 1 },
			function(data) {
				console.log('success');
				console.log(this);
				console.log(data);
			}
		);
	}]);
