Getting Started

The first thing to do is create a ViewModel in your javascript: 

Curvy.register.viewmodel('getting-started', function() {
   this.myTitle = 'My First Application';
   this.updateTitle = function() {
      this.myTitle = 'My First Binding';
   };
});

Next we must wire up the new functionality to the DOM in your html 

<div view-model="getting-started">
   <h1 data-format="{{myTitle}}"></h1>
   <button data-click="updateTitle">Change The Words</button>
</div>

Finally we tell Curvy to startup like so: 

Curvy.create();
