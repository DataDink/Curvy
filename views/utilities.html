Utilities

The utitlies service contains a small number of common functionalities used frequently by Curvy.

nothing Always undefined.

is(value, type) Determines if a value is of a certain type. This will compare the value by constructor or type string.

readonly(obj, member, getter) Sets a read-only property getter on the object.

constant(obj, member, value) Sets a read-only value on the object.

path(obj, path) Resolves a value starting from the object following the specified dot "." separated path.

watch(obj, path, callback) Listens to an observable and the descendant observable objects referenced by the specified dot "." path.
