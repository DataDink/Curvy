Utilities

The utitlies service contains a small number of common functionalities used frequently by Curvy.

nothing Always undefined.

is(value, type) Determines if a value is of a certain type. This will compare the value by constructor or type string.

readonly(obj, member, getter) Sets a read-only property getter on the object.

constant(obj, member, value) Sets a read-only value on the object.

path(obj, path) Resolves a value starting from the object following the specified dot "." separated path.

watch(obj, path, callback) Listens to an observable and the descendant observable objects referenced by the specified dot "." path.

clone(obj) Creates a shallow copy of an object

merge(from, to) Creates a shallow copy of the "to" object with non-conflicting members from the "from" object.

toarray(collection) Attempts to convert a collection-type object into an actual array.

distinct(array) Removes duplicate items from an array.

domdepth(element) Counts how far nested an element is in the document.

domsort(array) Sorts an array of elements by their nested depth in the document.

domquery(element, selector) Performs a querySelectorAll from the element and includes the element in the results if it matches the selector

indom(element) Returns true if the element is currently nested anywhere in the DOM
